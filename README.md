Запуск проекта локально:

1. Клонировать API: 
    git clone https://github.com/Renat2202/movies-explorer-api.git
    
2. Запустить API:
    npm run start
    
3. Клонировать Frontend:
    git clone https://github.com/Renat2202/movies-explorer-frontend.git
    
4. Запустить Frontend:
    npm run start
